如果你已经有最终 .pcd 文件，直接这样跑：

roslaunch pcd_to_2d_grid_map pcd_to_2d_grid_map.launch \
  mode:=pcd \
  pcd_path:=/home/unitree/slam_ws/maps/all_raw_points.pcd \
  output_dir:=/home/unitree/slam_ws/maps \
  map_name:=g1_nav_map \
  resolution:=0.04 \
  z_min:=0.1 \
  z_max:=1.5 \
  voxel_leaf_size:=0.05 \
  padding_radius:=0.12 \
  min_points_per_cell:=1

如果你想直接从 FAST-LIVO2 发布的全局点云话题生成：

roslaunch pcd_to_2d_grid_map pcd_to_2d_grid_map.launch \
  mode:=topic \
  pointcloud_topic:=/Laser_map \
  output_dir:=/home/unitree/maps \
  map_name:=g1_nav_map

生成后你会得到：

/home/unitree/maps/g1_nav_map.pgm
/home/unitree/maps/g1_nav_map.yaml

然后可以直接用 map_server 验证：

rosrun map_server map_server /home/unitree/maps/g1_nav_map.yaml

在 RViz 里显示 /map 即可。
